## 🔄 雙機無縫交接 SOP 指南

當您要從 **電腦 A** (例如 Mac mini) 換到 **電腦 B** (例如 Macbook) 時，請遵循以下步驟：

### 1. 在 電腦 A 結束工作時：
在 Terminal 中執行存檔三步驟：
```bash
git add .
git commit -m "checkpoint: 完成目前開發，準備換電腦交接"
git push
```

### 2. 在 電腦 B 準備開工時：
1. 打開 Terminal，先讀取最新存檔：
   ```bash
   git pull
   ```
2. 打開 Google Antigravity IDE，直接在對話框中輸入：
   > 「*我剛剛換電腦了，請讀取專案內的 walkthrough.md 檔案以接續上一台電腦的進度，並告訴我接下來該做什麼。*」
3. AI 讀取 `walkthrough.md` 後，就會 100% 讀懂您的前情提要，無縫接軌繼續工作！
