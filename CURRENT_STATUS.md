## 📅 當前專案最新狀態 (截至 2026-07-15)

* **最新版本號**：`v1.07.18`
* **最新完成功能**：
  1. **實作「自訂分群與議題」初始化上傳防呆**：
     - **本地資料保護機制**：為避免新裝置連線全新 Sheets 工作表時因空資料覆蓋本地快取，新增初始化判定。若雲端返回空資料而本地有自訂分群或常規議題，將自動觸發把本地資料推送到雲端，確保原有資料 100% 不丟失。
  2. **實作「常規議題」與「分群管理」跨裝置雲端同步**：
     - **後端擴增工作表**：在雲端試算表新增 `Issues` 與 `Groups` 工作表，用作跨裝置存儲。
     - **即時自動雙向同步**：新增或刪除議題/分群時，系統會立即在背景自動與雲端 Sheets 對應工作表雙向同步（GET/POST）。
     - **多裝置資料一致**：大幅提升多終端使用者自訂分群與常規議題的體驗一致性。
  2. **隱藏 `[團體]` 前綴優化展示**：
     - **過濾顯示姓名**：當主看板卡片、待辦提醒以及今日到訪卡片渲染客戶姓名時，會自動隱藏 `[團體] ` 前綴，直接顯示實際名稱（如「科林研發...」），徹底解決因前綴佔用字寬導致名稱被截斷的視覺 Bug。
     - **優雅標註識別**：為團體案件卡片上的姓名特別新增了「虛線底線」視覺樣式，可在免去冗長前綴的同時，一眼識別出這是約訪主紀錄。
  2. **團體成員「即時關鍵字搜尋選取」優化**：
     - **極速自動完成搜尋框**：將原本靜態的成員下拉選單重構為具有 Autocomplete 動態過濾的搜尋輸入框。當輸入姓名關鍵字時，會即時在下方滑出符合條件的選單。
     - **優雅選取與防呆**：支援模糊匹配與點擊選入，同時兼具失焦（Blur）延遲自動收納，避免操作干擾，且加入了空白防呆機制。
  2. **實作「團體/家庭約訪管理」(方案 A)**：
     - **快速勾選建立**：新增客戶時可勾選「這是團體/家庭約訪」，系統會自動加上 `[團體] ` 前綴，用作彙整主紀錄。
     - **成員關聯組件**：在 `[團體]` 案件的客戶 Drawer 中新增「參與成員名單」小工具。能快速下拉選擇案件成員、加入/移除，並將成員列表以藍色小膠囊標籤呈現。
     - **極速切換 Drawer**：點擊參與成員名單中的名字，可直接在 Drawer 中秒切到該名單成員的資料進行紀錄與推進，完美解決多人團約時各成員進度與備忘維護的需求。
     - **無痛資料庫序列化**：將 `groupMembers` 成員資料無縫壓縮序列化進現有的 `聯絡資訊` JSON 欄位中，達成跨裝置雙機完全同步，**100% 不需修改 Apps Script 後端代碼與 Google Sheets 欄位定義**。
  2. **新增「電訪」約訪類型**：
     - 在「新增客戶與議題」對話框以及案件細節側欄（Drawer）的「訪談類型」按鈕組中，新增了「電訪」選項。
     - 在案件卡片上，為電訪類型新增了亮紫色徽章（顯示「電」）以及對應的 CSS 樣式 `.source-badge.call-visit`，以符合台灣保險與金融市場的電訪作業情境。

     - **移除手動雲端同步功能**：全面刪除「雲端同步」按鈕與相關代碼，完全依靠點擊儲存時的 100% 自動即時背景同步。
     - **編輯按鈕移至 Header**：點選「新增流程」或「編輯流程」時，右上角的 `＋ 新增流程` 按鈕會動態被替換為 `儲存定義` 和 `取消`；儲存或取消後自動回復。同時刪除了原本編輯器下方多佔空間的按鈕列，徹底釋放垂直空間，避免介面擁擠。
     - **全面清除 ICON 表情符號**：移除了通知面板/維護中心裡的所有 ICON 標記（如 💼, 🛡️, 📋, 🧭, 🎴, 🚀 等），使介面回歸純文字，降低視覺干擾。

  2. **銷售流程定義維護功能**：在「專業準備與工具」維護中心內新增「🧭 銷售流程定義」頁籤。包含「階段」、「任務」、「用意」、「文件」等四個欄位的展示與編輯器。支持獨立拖曳 Drag & Drop 調整排序，並具備本地快取及雲端 API 同步載入儲存。
  2. **增員輔導管理雙欄視角重構**：將增員管理主面板重構為雙欄 master-detail 設計。左側頂部設置三大方塊準備工作按鈕，下方為增員清單列表；右側則為動態面板，點擊準備工作顯示其編輯器，點擊增員對象則顯示其「🧭 增員路徑」基本資料與面談進度及「📝 輔導紀錄」頁籤，扁平化整合大幅降低操作阻力。

## 🔄 雙機無縫交接 SOP 指南

當您要從 **電腦 A** (例如 Mac mini) 換到 **電腦 B** (例如 Macbook) 時，請遵循以下步驟：

### 1. 在 電腦 A 結束工作時：
在 Terminal 中執行存檔三步驟：
```bash
git add .
git commit -m "checkpoint: 完成目前開發，準備換電腦交接"
git push
```

### 2. 在 電腦 B 準備開工時：
1. 打開 Terminal，先讀取最新存檔：
   ```bash
   git pull
   ```
2. 打開 Google Antigravity IDE，直接在對話框中輸入：
   > 「*我剛剛換電腦了，請讀取專案內的 walkthrough.md 檔案以接續上一台電腦的進度，並告訴我接下來該做什麼。*」
3. AI 讀取 `walkthrough.md` 後，就會 100% 讀懂您的前情提要，無縫接軌繼續工作！
