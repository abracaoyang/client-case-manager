# 📊 Google Sheets 雲端資料庫串接指南

本系統支援將所有客戶案件進度資料**即時背景備份**與**跨裝置同步**至您的 Google Drive 雲端試算表。

請按照以下步驟完成雲端資料庫的建立與部署串接：

---

## 🛠️ 第一步：在雲端硬碟中建立 Google 試算表
1. 登入您的 Google 帳號，並進入您的 **Google 雲端硬碟 (Google Drive)**。
2. 進入您此專案的資料夾：`我的雲端硬碟 > 客戶案件管理系統`。
3. 點擊左上角的 **「新增 (＋)」 > 「Google 試算表」** 建立一個新的空白試算表。
4. 將試算表命名為：**`客戶案件資料庫`**（或其他您喜愛的名稱）。

---

## 💻 第二步：部署 Google Apps Script 後端程式碼
1. 在剛剛建立的試算表內，點擊頂部選單的 **「延伸功能」 > 「Apps Script」**。
2. 系統會自動為您開啟 Apps Script 編輯器，請將預設程式碼（包含 `myFunction()`）全部清空。
3. 開啟本機資料夾中的 **[code.gs](file:///Users/oyangyan/Library/CloudStorage/GoogleDrive-abracaoyang@gmail.com/我的雲端硬碟/客戶案件管理系統/code.gs)** 檔案，並將其全部內容**複製**。
4. 貼回 Apps Script 編輯器中，點擊編輯器上方的 💾 (儲存) 按鈕。

---

## 🚀 第三步：部署為網頁應用程式 (Web App)
1. 點擊 Apps Script 編輯器右上角的 **「部署」 > 「新增部署」** 按鈕。
2. 在彈出的視窗中，點擊齒輪圖示旁的 **「選取類型」**，選擇 **「網頁應用程式」**。
3. 填入設定：
   - **說明**：`客戶案件管理系統 API`
   - **預行身份**：選擇 **「我」 (您的 Google 帳戶)**
   - **誰有權限存取**：選擇 **「任何人」** (此項為確保本機 HTML 能正確發送 API 請求)
4. 點擊 **「部署」**。
5. 此時 Google 會要求授權，請點擊 **「授予存取權」**，選取您的 Google 帳號，點選 **「進階」** ➡️ 點擊 **「前往『未命名專案』(不安全)」**，最後點選 **「允許」**。
6. 部署成功後，視窗中會提供一串 **「網頁應用程式的網址」** (例如 `https://script.google.com/macros/s/.../exec`)。
7. 請點擊 **「複製」** 將此 API 網址複製下來。

---

## 🔗 第四步：與本機管理系統進行連線綁定
1. 雙擊開啟您本機的 **[index.html](file:///Users/oyangyan/Library/CloudStorage/GoogleDrive-abracaoyang@gmail.com/我的雲端硬碟/客戶案件管理系統/index.html)** 網頁。
2. 點擊網頁右上角的 **「⚙️ 系統設定」** 按鈕。
3. 在彈出視窗中：
   - 將模式切換為：🔴 **「雲端連線模式 (同步 API)」**。
   - 在下方 **「雲端 API 連線網址 (Apps Script Web App URL)」** 輸入框中，**貼上**您在第三步複製的部署網址。
4. 點擊右側的 **「測試連線」** 按鈕，若出現「連線測試成功！」綠色 Toast，代表串接完美成功！
5. 點擊 **「儲存設定」**。

---

## 💡 使用小撇步與防呆機制
* **自動初始化**：若您剛建立空白試算表，當系統第一次與其同步時，`code.gs` 會自動在 `Sheet1` 內為您建立所有所需的欄位表頭。
* **背景自動同步與佇列**：當您在網路斷線、離線模式下操作時，修改資料會自動存在本機。當您下次開啟網頁、儲存設定或連線恢復時，系統會跳出提示，並自動在背景把積壓的資料同步回 Google 試算表，確保不遺失任何進度！
